# Legistar Web API

Base URL: `https://webapi.legistar.com/v1/{client}`

## Key Endpoints

| Endpoint | Description |
|----------|-------------|
| `/Bodies` | Boards, commissions, and councils |
| `/Events` | Meetings and hearings |
| `/EventItems` | Agenda items for a given event |
| `/Matters` | Legislative matters (ordinances, resolutions, etc.) |
| `/Persons` | Council members and staff |
| `/Votes` | Voting records |
